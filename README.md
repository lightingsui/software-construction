## 简介

软件构造实验要求，编码电厂管理系统。此系统采用JavaSwing组件开发，mysql5.7，jdk1.8，IDEA，满足课设要求。

在src下存在一个lib包，将其作为全局库即可运行，sql文件可直接从Navicat导入。
